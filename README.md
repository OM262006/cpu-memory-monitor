# System Health Checker 

A command-line tool to monitor system performance using Python.

## Features
- Displays CPU usage
- Displays memory usage
- Displays disk usage
- Classifies system load (Idle, Normal, High, Critical)

## Tech Used
- Python
- psutil library

##  How to Run

bash
pip install psutil
python main.py


>> Example Output

CPU Usage: 25%
Memory Usage: 60%
Disk Usage: 70%
